# 우리집 — 가족 스케줄 공유 앱

가족 캘린더, 할일·장보기·위시리스트 공유, 실시간 알림을 갖춘 웹 앱입니다.
빌드 도구 없이 순수 HTML/CSS/JS + Firebase로 동작해서, Firebase 프로젝트만 연결하면 바로 씁니다.

## 1. Firebase 프로젝트 만들기 (5분)

1. https://console.firebase.google.com 접속 → "프로젝트 추가"
2. 왼쪽 메뉴 **빌드 → Authentication** → "시작하기" → **로그인 방법**에서 "이메일/비밀번호" 사용 설정
3. 왼쪽 메뉴 **빌드 → Firestore Database** → "데이터베이스 만들기" → 테스트 모드로 시작 (위치는 asia-northeast3 서울 추천)
4. Firestore 화면의 **규칙(Rules)** 탭에 이 프로젝트의 `firestore.rules` 내용을 붙여넣고 게시 (가족 데이터가 외부에 노출되지 않도록 하는 보안 규칙입니다)
5. 왼쪽 위 톱니바퀴 → **프로젝트 설정 → 내 앱 → 웹 앱 추가(</>)** → 앱 등록 후 나오는 `firebaseConfig` 값을 복사
6. `firebase-config.js` 파일을 열어 복사한 값을 붙여넣기

## 2. 로컬에서 미리보기

인터넷이 연결된 아무 정적 서버로 이 폴더를 열면 됩니다. 예:
```
npx serve .
```

## 4. 사용법

- 회원가입 후, 가족 중 한 명이 "가족 만들기"로 초대 코드를 생성
- 나머지 가족은 "참여하기"에서 그 코드를 입력
- **캘린더**: 날짜를 눌러 일정 추가 (담당자 지정 가능). 그날 마감인 할일도 일정과 함께 표시돼요
- **요청**: 할일·장보기·위시리스트를 한 탭에 모아뒀어요. 맨 위 현황 칩(할일/장보기/위시 남은 개수)이나 아래 토글로 원하는 섹션을 골라 입력해요
  - 할일: 집안일·심부름 등을 등록하고 담당자를 지정. 완료되면 동그라미를 눌러 체크. "전체/내 할일/완료됨"으로 필터링 가능
  - 장보기: 사고 싶은 생필품을 아무나 추가 → 장 보러 가는 사람이 산 뒤 체크. 항목을 두 번 누르면 삭제
  - 위시: 먹고 싶은 것·받고 싶은 선물을 적어두면 가족이 함께 봐요
- **가족 탭 → 생일·기념일**: 생일이나 기념일을 등록하면 캘린더에 매년 자동으로 표시되고, 3일 전 / 당일에 알림
- 종 아이콘을 눌러 브라우저 알림 허용 시, 다른 가족이 일정을 추가하거나 생일이 다가오면 실시간 알림

### 이미 예전 버전을 배포했다면

Firestore **규칙(Rules)** 탭에 이 프로젝트의 `firestore.rules` 최신 내용을 다시 붙여넣고 게시해주세요. 할일·장보기·기념일 데이터를 위한 권한이 추가되었어요.
