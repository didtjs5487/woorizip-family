설정이 하나 남았어요

이 앱은 Firebase를 백엔드로 사용해요. firebase-config.js 파일을 열어 본인의 Firebase 프로젝트 값을 붙여넣으면 바로 실행돼요.

  1. firebase.google.com/console 접속 후 새 프로젝트 생성
  2. 빌드 → Authentication → 익명(Anonymous) 로그인 사용 설정
  3. 빌드 → Firestore Database 만들기 (테스트 모드로 시작)
  4. 프로젝트 설정 → 내 앱 → 웹 앱 추가 → 설정 값 복사
  5. firebase-config.js에 붙여넣고 저장
